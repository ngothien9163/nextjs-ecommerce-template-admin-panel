📊 Cách SEO Page Types và SEO Pages hoạt động:
🎯 SEO Page Types (7 loại):
page: Trang tĩnh (trang chủ, giới thiệu)
product: Trang sản phẩm (SEO cao nhất)
category: Trang danh mục (từ khóa tập trung)
user: Trang người dùng (hồ sơ, đánh giá)
system: Trang hệ thống (404, sitemap)
blog: Trang blog, tin tức
landing: Trang đích marketing

🔍 SEO Pages (Mỗi trang web):
Mỗi trang sẽ có bản ghi SEO riêng với:
Meta tags: Title, description, keywords
Open Graph: Facebook, social media
Twitter Cards: Twitter optimization
Schema markup: JSON-LD structured data
Core Web Vitals: LCP, FID, CLS, INP, TTFB
AI/ML metrics: Relevance score, ranking factors
E-E-A-T: Experience, Expertise, Authoritativeness, Trust
Performance: Page load time, mobile score, accessibility

⚡ Cách tối ưu cho từng loại trang:
🏠 Trang chủ:
Title: 50-60 ký tự, chứa từ khóa chính
Meta description: 150-160 ký tự
Schema: WebSite type
Từ khóa: "shop online", "thời trang"

�� Trang sản phẩm:
Title: Tên sản phẩm + thông số + thương hiệu
Meta description: Mô tả + lợi ích + call-to-action
Schema: Product type với giá, thương hiệu
Core Web Vitals: LCP < 2.5s, FID < 0.1s

�� Trang danh mục:
Title: Tên danh mục + sản phẩm chính + thương hiệu
Meta description: Liệt kê sản phẩm tiêu biểu
Schema: CollectionPage type
Từ khóa: Tập trung vào danh mục

📝 Trang blog:
Title: Chủ đề + năm + nguồn uy tín
Meta description: Tóm tắt + lợi ích
Schema: Article type với headline, author
Nội dung: Tối thiểu 1000 từ, dễ đọc

�� Các yếu tố SEO 2025-2026+:
Core Web Vitals 2.0: LCP, FID, CLS, INP, TTFB
AI & ML SEO: Relevance score, ranking factors
E-E-A-T: Experience, Expertise, Authoritativeness, Trust
Voice & Visual Search: Voice optimization, visual data
Privacy & Compliance: GDPR, CCPA, privacy signals
Future Metrics: Quantum SEO, Neural networks, BCI
